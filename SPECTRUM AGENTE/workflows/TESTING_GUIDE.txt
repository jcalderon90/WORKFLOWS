🚀 CÓMO EMPEZAR (3 OPCIONES)
═══════════════════════════════════════════════════════════════════════════

OPCIÓN 1: Menú interactivo (RECOMENDADO PARA NO PROGRAMADORES)
────────────────────────────────────────────────────────────────
$ cd "Agente Unificado"
$ ./scripts/run_tests.sh

→ Te mostrará un menú para elegir qué test ejecutar

───────────────────────────────────────────────────────────────────────────

OPCIÓN 2: Comandos directos (PARA DESARROLLADORES)
────────────────────────────────────────────────────────────────
$ cd "Agente Unificado"

# Ver qué tests hay
$ python3 scripts/test_agent.py --list

# Ejecutar un test rápido
$ python3 scripts/test_agent.py --scenario saludo_inicial

# Ejecutar la conversación completa
$ python3 scripts/test_agent.py --scenario happy_path_completo

# Ejecutar TODOS (10 tests, ~10 minutos)
$ python3 scripts/test_agent.py

───────────────────────────────────────────────────────────────────────────

OPCIÓN 3: Script bash directo
────────────────────────────────────────────────────────────────
$ cd "Agente Unificado"
$ ./scripts/run_tests.sh quick      # Test rápido
$ ./scripts/run_tests.sh happy      # Happy path
$ ./scripts/run_tests.sh all        # Todos


📋 LOS 10 ESCENARIOS
═══════════════════════════════════════════════════════════════════════════

1. saludo_inicial
   → Valida bienvenida sin datos previos
   ⏱ ~30 segundos

2. happy_path_completo ⭐ RECOMENDADO
   → Conversación completa: saludo → datos → proyecto → consulta
   ⏱ ~3-5 minutos

3. lead_sin_datos
   → Verifica que lead_collector se active cuando faltan datos
   ⏱ ~30 segundos

4. fuera_de_contexto
   → Valida guardrail (rechaza código, recetas, medicina, etc.)
   ⏱ ~30 segundos

5. mencion_proyecto_directo
   → Usuario menciona proyecto en primer mensaje
   ⏱ ~30 segundos

6. solicita_asesoria
   → Usuario pide hablar con asesor humano
   ⏱ ~1-2 minutos

7. multiples_proyectos
   → Maneja ambigüedad ("zona 15" → PVV o PSB)
   ⏱ ~30 segundos

8. canal_instagram
   → Funciona igual en Instagram que en WhatsApp
   ⏱ ~30 segundos

9. rsvp_flow
   → Usuario quiere agendar una visita/cita
   ⏱ ~3-5 minutos

10. modismos_guatemaltecos
    → Entiende "simón", "va", "nel", etc.
    ⏱ ~1-2 minutos
